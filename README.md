# 🌍 Synq - Connect, Chat, Call Instantly

A Real-Time Cross-Language & Cultural Exchange Platform built using the
MERN Stack.

------------------------------------------------------------------------

## 🚀 About The Project

**Synq** is a full-stack MERN application designed to connect users
across different countries for:

-   🌎 Cultural Exchange\
-   🗣️ Language Learning\
-   💬 Real-Time Chat\
-   📹 Live Video Calling\
-   👥 Social Networking (Follow / Following System)

This project is developed as an MCA Final Semester Major Project and is
inspired by the YouTube tutorial:

**"MERN Stack Project: Video Calling Realtime Chat App & Social App" --
Codesistency**

------------------------------------------------------------------------

## ⚙️ Installation Guide

### 1️⃣ Clone Repository

git clone repo-link(https://github.com/username/repo_name)

### 2️⃣ Backend Setup

cd server\
npm install

Create `.env` file:

PORT=5000\
MONGO_URI=your_mongodb_connection_string\
JWT_SECRET=your_secret_key

Run Backend: npm run dev

### 3️⃣ Frontend Setup

cd client\
npm install\
npm start
